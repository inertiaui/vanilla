# Installation

Install the package via npm:

```bash
npm install @inertiaui/vanilla
```

## Usage

Import the functions you need:

```typescript
import {
    createDialog,
    lockScroll,
    onEscapeKey,
    onClickOutside,
    generateId,
    except,
    only,
} from '@inertiaui/vanilla'
```

## TypeScript

The package includes full TypeScript support. You can import types directly:

```typescript
import type {
    CleanupFunction,
    DialogOptions,
    Dialog,
    FocusTrapOptions,
    ClickOutsideOptions,
    EscapeKeyOptions,
} from '@inertiaui/vanilla'
```
